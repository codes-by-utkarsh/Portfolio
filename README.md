# Portfolio - Utkarsh Srivastava (0xutkarsh)

A minimalistic, cyber-themed portfolio website built with React and Vite.

## 🚀 Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Locally:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## 🛠️ Tech Stack

-   **React**: UI Library
-   **Vite**: Build Tool
-   **CSS Modules / Vanilla CSS**: Styling (Minimalist Cyber Theme)

## 🎨 Features

-   **Hero Section**: Introduction and Profile Image.
-   **About**: Bio and Current Status.
-   **Skills**: Comprehensive list of technical skills.
-   **Services**: Pricing for pentesting and simulated Payment Gateway.
-   **Contact**: Contact form and links.

## 🔐 Security Note

This is a client-side application. The payment gateway is currently a **simulation** for demonstration purposes as requested. For real payments, backend integration with Stripe/PayPal is required.
